# ADR-0013 — Intraday Auto Card Scope

| 항목 | 내용 |
| --- | --- |
| 문서 종류 | ADR (Architecture Decision Record) |
| 제품명 | Market Manipulator Survival |
| 상태 | Accepted |
| 날짜 | 2026-06-13 |
| 관련 문서 | PRD v0.1.4, ADR-0006, ADR-0007, ADR-0008, ADR-0011, ADR-0012 |
| 결정 범위 | MVP 장중 자동 카드 목록, 수동 액션과의 역할 분리, 자동 카드 성장 구조 |

---

## Context

MVP 장중 수동 액션은 4개로 확정되었다.

```text
유동성 공급
가격 추진
과열 해소
포지션 정리
```

수동 액션은 플레이어가 지금 당장 누르는 대응 버튼이다.

반면 Vampire Survivors식 재미를 만들려면 장중 자동 카드는 플레이어가 반복 입력하는 버튼이 아니라, 선택한 빌드에 따라 주기적으로 발동되는 성장 요소여야 한다.

---

## Decision

MVP 장중 자동 카드는 8개로 제한한다.

1. 관심 신호
2. 유동성 순환
3. 가격 지지
4. 변동성 흡수
5. 뉴스 증폭
6. 감시 완충
7. 경쟁 견제
8. 정리 루틴

자동 카드는 플레이어가 직접 누르는 수동 액션과 달리, 장중에 주기적으로 발동되거나 지속 효과를 제공하는 빌드 요소다.

플레이어는 장중 일정 조건마다 자동 카드 선택지를 받고, 새 카드를 획득하거나 기존 카드를 강화할 수 있다.

---

## Card Roles

| 카드 | 담당 지표 | 역할 |
| --- | --- | --- |
| 관심 신호 | 개인 참여도 | 시장 관심을 끌어 가격 반응성을 높이는 성장 카드 |
| 유동성 순환 | 시장 유동성 | 장중 시장 흐름을 활발하게 만들어 가격이 더 잘 움직이게 하는 카드 |
| 가격 지지 | 가격 방어 | 악재나 패닉 상황에서 가격 붕괴를 막는 방어 카드 |
| 변동성 흡수 | 변동성 | 과도한 흔들림을 줄여 목표 밴드 안착을 돕는 안정화 카드 |
| 뉴스 증폭 | 뉴스 효과 | 그날의 시장 분위기에 올라타는 고위험/고보상 카드 |
| 감시 완충 | 감시도 | 고위험 운용을 조금 더 오래 버티게 해주는 방어적 카드 |
| 경쟁 견제 | 경쟁 압박 | 다른 시장 압력이나 경쟁 데스크의 방해를 줄이는 카드 |
| 정리 루틴 | 보유 비중/예산 회수 | 마감 전에 과도한 보유 비중을 줄이고 정산 리스크를 낮추는 카드 |

---

## Auto Card Growth

각 자동 카드는 Lv.1~Lv.3까지 성장할 수 있다.

MVP에서는 레벨업 시 다음 중 하나만 개선한다.

1. 효과량 증가
2. 발동 주기 단축

복잡한 분기 업그레이드, 카드 진화, 조합 시너지는 P1 이후로 둔다.

---

## Acquisition

MVP 자동 카드 획득 방식은 다음 방향으로 둔다.

1. Day 시작 시 기본 자동 카드 1개를 보유한다.
2. 장중 일정 조건마다 카드 선택 보상이 발생한다.
3. 선택지는 3개로 제한한다.
4. 선택한 카드는 새 카드 획득 또는 기존 카드 레벨업으로 처리한다.

정확한 보상 발생 조건, 발동 주기, 효과량, 레벨별 수치는 SRS/SDD에서 확정한다.

---

## Consequences

### Positive

1. 수동 액션과 자동 카드의 역할이 분리된다.
2. 장중 플레이에 Vampire Survivors식 성장 선택이 생긴다.
3. 8개 카드가 개인 참여도, 유동성, 가격 방어, 변동성, 뉴스 효과, 감시도, 경쟁 압박, 보유 비중/예산 회수를 고르게 다룬다.
4. Lv.1~Lv.3 구조로 성장 재미를 주면서 MVP 복잡도를 제한한다.

### Negative

1. 자동 카드 8개는 밸런싱 대상이 된다.
2. 수동 액션과 효과가 겹치지 않도록 UI 설명이 필요하다.
3. 카드 선택 보상 타이밍이 너무 잦으면 장중 흐름이 끊길 수 있다.

### Mitigations

1. 정확한 수치와 발동 주기는 SRS/밸런싱 단계로 넘긴다.
2. MVP에서는 카드 진화와 조합 시너지를 제외한다.
3. 선택지는 3개로 제한해 선택 피로를 줄인다.

---

## Alternatives Considered

### Auto cards without level-up

구현은 단순하지만 빌드 성장감이 약해진다.  
MVP에서도 Lv.1~Lv.3까지만 허용해 최소 성장감을 둔다.

### More than 8 auto cards

빌드 다양성은 늘지만 밸런싱과 설명 부담이 커진다.  
MVP에서는 핵심 지표별 대표 카드 8개로 제한한다.

### Complex evolution or synergy

뱀서식 재미는 강해질 수 있지만 MVP 범위가 커진다.  
카드 진화, 조합 시너지, 분기 업그레이드는 P1 이후로 둔다.

---

## PRD Impact

PRD v0.1.4는 다음을 반영한다.

1. MVP 장중 자동 카드 8개를 명시한다.
2. 자동 카드는 주기 발동 또는 지속 효과를 제공하는 빌드 요소로 정의한다.
3. 자동 카드는 Lv.1~Lv.3까지만 성장한다.
4. 레벨업은 효과량 증가 또는 발동 주기 단축 중 하나만 적용한다.
5. 복잡한 분기 업그레이드, 카드 진화, 조합 시너지는 P1 이후로 둔다.
