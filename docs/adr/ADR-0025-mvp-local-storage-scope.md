# ADR-0025 — MVP Local Storage Scope

| 항목 | 내용 |
| --- | --- |
| 상태 | Accepted |
| 날짜 | 2026-06-13 |
| 관련 문서 | PRD v0.1.5, ADR-0001, ADR-0020, ADR-0024 |

## Context

MVP는 브라우저에서 실행되는 싱글플레이어 오프라인 게임이다.

Run Seed와 같은 조건 재시작이 도입되면서 최소한의 로컬 저장 범위가 필요하다. 다만 상세 로그, 리플레이, 클라우드 저장, 온라인 랭킹은 MVP의 핵심 재미 검증에 직접 필요하지 않고 구현 범위를 늘린다.

## Decision

MVP에서는 로컬 저장만 사용한다.

저장 대상은 다음으로 제한한다.

1. 진행 중 Run 상태
2. 최근 Final Settlement 결과
3. 최고 Final 등급
4. 최고 누적 수익

상세 플레이 로그, 리플레이 데이터, 클라우드 저장, 온라인 랭킹은 MVP에서 제외한다.

## Consequences

MVP는 중단 후 이어하기와 기본 기록 확인만 지원한다.

플레이어는 최근 결과와 최고 성과를 확인할 수 있지만, 상세 플레이 분석이나 리플레이 검토는 할 수 없다.

저장 구조는 작게 유지되며, 클라우드 동기화나 서버 의존성이 없다.

정확한 저장 키, 데이터 구조, 마이그레이션 정책은 SRS/SDD 단계에서 정의한다.

## Alternatives Considered

### Detailed Play Log in MVP

기각한다.  
분석과 밸런싱에는 유용하지만 MVP 플레이어 경험에 필요한 범위를 넘는다.

### Cloud Save or Online Ranking

기각한다.  
서버, 계정, 동기화, 보안 범위를 추가하므로 MVP에서는 제외한다.
